## Required Data Per Figure

| Figure | Main table used in reference | Required columns (minimum) |
|---|---|---|
| `pas_detect_performance.pdf` | `match_performance` | `tool`, `match_type`, `precision`, `recall`, `f1` |
| `pas_detect_{metric}_by_protocol.pdf` | `match_performance` | `protocol`, `tool`, `match_type`, `{metric}` |
| `pas_quantification_performance.pdf` | `pas_quantification_performance` | `tool`, `match_type`, `cor_pas`, `mape_pas`, `mape_pas_ct` |
| `pas_quantification_{metric}_by_protocol.pdf` | `pas_quantification_performance` | `protocol`, `tool`, `match_type`, `{metric}` |
| `apa_detect_performance.pdf` | `de_apa_performance` (filtered + aggregated) | `tool`, `match_type`, `precision`, `recall`, `f1` |
| `apa_detect_{metric}_by_protocol.pdf` | `de_apa_performance` (filtered + aggregated) | `protocol`, `tool`, `match_type`, `{metric}` |
| `filter_residuals_heatmap.pdf` | derived from `de_apa_performance` | `filter_type_1`, `filter_type_2`, `f1_residuals` |
| `max_residuals_vs_gt_recall_with_arrows.pdf` | derived from `de_apa_performance` | `filter_type_comb`, `f1_residuals`, `gt_recall` |

Notes for residual figures:

- Reference notebook derives:
  - `gt_recall = gt_count / 5000`
  - `f1_residuals` via per-group regression (`sample/tool/match_type`)
  - `filter_type_comb = filter_type_1 + "|" + filter_type_2`
