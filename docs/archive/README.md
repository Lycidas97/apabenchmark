# Archive

This directory keeps historical implementation notes and legacy scripts that
are not part of the current public reproduction entrypoints.

Archived files may contain obsolete paths, workstation-specific examples, or
intermediate decisions retained for provenance. Do not use them as setup or
execution instructions for the current workflow.

Archived debug reproducers, such as `locfit_failure_sample_repro/`, are kept to
document historical failure investigations and are not release validation tests.

Use the top-level `README.md`, `docs/full_repro_data_flow.md`, and numbered
`scripts/01_*` through `scripts/06_*` stage directories for the active workflow.
